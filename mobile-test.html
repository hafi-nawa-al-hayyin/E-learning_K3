🧪 Cara Testing Responsiveness

Opsi 1: Browser DevTools (Recommended)

  1. Tekan F12 atau Right-click → Inspect
  2. Klik ikon toggle device (Ctrl+Shift+M)
  3. Pilih device preset atau custom width
  4. Perhatikan perubahan layout saat resize

Opsi 2: Perangkat Fisik

  1. Buka aplikasi di smartphone Android/iOS
  2. Rotasi device untuk landscape mode
  3. Amati hamburger menu dan layout changes
  4. Test semua buttons dan form inputs

⚡ Tips Performa Mobile

  • Minimize CSS file dengan minifier untuk bandwidth lebih kecil
  • Optimize images dengan compression tools
  • Use lazy loading untuk images dan content
  • Cache assets untuk offline support (PWA)
  • Monitor Core Web Vitals dengan PageSpeed Insights
